# EEG Motor Imagery Prediction Application

A Streamlit application for classifying imagined left-hand vs right-hand fist
movements from EEG signals, using the
[PhysioNet EEG Motor Movement/Imagery Dataset](https://www.physionet.org/content/eegmmidb/1.0.0/).

Built for the **ARISE Summer Research Internship** technical challenge тАФ
*Towards Explainable Multimodal AI for Fatigue Detection Using EEG and Wearable Sensors*.

## Pipeline

### 1. Dataset

| Setting | Value |
|---------|-------|
| Dataset | PhysioNet EEG Motor Movement/Imagery (v1.0.0) |
| Subjects | S001тАУS040 (excluding known bad subjects: 88, 89, 92, 100, 104) |
| Runs | 4, 8, 12 (imagined left/right fist) |
| Annotations | T1 = left fist (class 1), T2 = right fist (class 2), T0 = rest (ignored) |

### 2. Preprocessing

| Step | Detail |
|------|--------|
| Channel standardisation | `mne.datasets.eegbci.standardize` + standard_1005 montage |
| Band-pass filter | 8тАУ30 Hz, zero-phase FIR (`firwin`) |
| Event extraction | T1 / T2 from EDF annotations via `mne.events_from_annotations` |
| Epoch window | **+0.5 s тЖТ +2.5 s** relative to cue onset (skips early VEP, captures sustained ERD) |
| Baseline correction | None |

### 3. Feature Extraction

Two complementary feature sets are extracted from each epoch:

#### Band-power features (for Random Forest / MLP)
Channels: 9 sensorimotor channels тАФ `FC3, FCz, FC4, C3, Cz, C4, CP3, CPz, CP4`

For each channel, Welch PSD тЖТ mean power in each band тЖТ log transform:

| Band | Range |
|------|-------|
| Mu / Alpha | 8тАУ13 Hz |
| Beta | 13тАУ30 Hz |

Additionally, a **C3/C4 laterality index** `(C4 тИТ C3) / (C4 + C3)` is appended
for each band тАФ the primary biomarker for left vs. right hand motor imagery.

Total: `9 channels ├Ч 2 bands + 2 laterality = 20 features` per epoch.

#### Spatial filter features (for CSP-LDA / Riemannian models)
All 64 EEG channels are used:

- **CSP-LDA**: Common Spatial Patterns (4 components, Ledoit-Wolf regularisation) тЖТ Linear Discriminant Analysis
- **Riemannian MDM**: Ledoit-Wolf covariance per epoch (9├Ч9 SPD matrix) тЖТ Minimum Distance to Mean on the Riemannian manifold
- **Riemannian TS-LDA**: Same covariance тЖТ Tangent Space projection тЖТ LDA (shrinkage)

### 4. Models

| Model | Input | Method | Within-subject Accuracy |
|-------|-------|--------|------------------------|
| `random_forest` | Band-power (20 feat.) | StandardScaler тЖТ RandomForest (500 trees) | 0.575 |
| `mlp` | Band-power (20 feat.) | StandardScaler тЖТ MLP (256-128-64, early stopping) | 0.494 |
| `csp_lda` | Raw epochs (64 ch.) | CSP (4 components, Ledoit-Wolf) тЖТ LDA | **0.619** |
| `riemann_mdm` | Raw epochs (64 ch.) | Covariances (Ledoit-Wolf) тЖТ MDM | 0.511 |
| `riemann_ts_lda` | Raw epochs (64 ch.) | Covariances тЖТ Tangent Space тЖТ LDA | **0.619** |

### 5. Evaluation

| Setting | Value |
|---------|-------|
| Strategy | Within-subject 80/20 split per subject, predictions aggregated across all subjects |
| Cross-validation | 5-fold stratified CV on pooled data (cross-subject reference) |
| Metrics | Accuracy, weighted F1, confusion matrix, classification report |

> **Note on evaluation:** Within-subject accuracy reflects individual-level performance
> (standard BCI evaluation). The uploaded-file accuracy (e.g. 80% on S001R04.edf)
> is higher because the model has seen similar data from that subject during training.

Results are saved to `models/model_metadata.json` and `results/offline_model_results.csv`.

## Run Locally

### Prerequisites

- Python 3.10тАУ3.13
- ~1 GB free disk space (for 40 subjects of PhysioNet data)

### 1. Clone and install

```bash
git clone https://github.com/Sandyshor/EEG-Motor-Imagery-Prediction.git
cd EEG-Motor-Imagery-Prediction

python -m venv .venv
# Windows:
.venv\Scripts\activate
# macOS/Linux:
source .venv/bin/activate

pip install -r requirements.txt
```

### 2. Train models offline (optional тАФ models already committed)

```bash
python train_models.py
```

Downloads S001тАУS040 (cached in `./data/`), trains all 5 models, saves to `models/`.
Expected runtime: 30тАУ90 minutes.

### 3. Launch the app

```bash
streamlit run app.py
```

Open `http://localhost:8501` in your browser.

### 4. Try it

1. Download a sample EDF тАФ subject S001 run 4:
   `https://physionet.org/files/eegmmidb/1.0.0/S001/S001R04.edf`
2. Upload it using the **Upload EDF file** widget in the sidebar.
3. Select `csp_lda` or `riemann_ts_lda` for best accuracy.
4. Inspect the visualisations, offline summary, and per-epoch predictions.

## Deploy on Streamlit Community Cloud

1. Push the repo to GitHub (models already committed).
2. Go to [share.streamlit.io](https://share.streamlit.io), sign in, create a new app
   pointing at `app.py` on the `main` branch.
3. Wait for the build (~3тАУ5 min), then paste the public URL into this README.

> The EEG dataset is **not** needed at deployment тАФ the app only reads the pre-trained
> `.pkl` files and processes uploaded EDF files on the fly.
